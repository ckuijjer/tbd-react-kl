# tbd-react

A simple image gallery

## Prerequisites

* [Node.js](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/en/)

## To start

* `yarn` - install dependencies
* `yarn start` - start the webpack dev server
  * `npm start` - start the webpack dev server using npm
